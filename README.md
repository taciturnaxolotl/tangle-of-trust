# The tangled web of trust

![screenshot of the app](https://l4.dunkirk.sh/i/f7XrvO_YwM2a.webp)

I made this right after [tangled launched](https://blog.tangled.org/vouching/) a vouching system. I was really curious how many people had started using it and how the graph had built over time so I started scrapping firehose data and then built a little force graph visulization! While I was at it I also scrapped historical follow data and added that as well :)

The canonical repo for this is hosted on tangled over at [`dunkirk.sh/tangle-of-trust`](https://tangled.org/dunkirk.sh/tangle-of-trust)
